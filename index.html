Core Components

Every computer is built from specialized hardware components, each performing a critical role in the system.

Processing

CPU — Central Processing Unit

The brain of the computer. Executes billions of instructions per second using billions of transistors etched onto silicon. Modern CPUs feature multiple cores, hyper-threading, and branch prediction to maximize throughput.

RAM — Random Access Memory

Ultra-fast temporary storage that holds running programs and active data. The CPU can read and write RAM millions of times faster than storage drives. Volatile — data is lost when power is removed.

Under the Hood

How a Computer Works

Every computation follows the same fundamental cycle, billions of times per second.

1

Fetch

The CPU fetches the next instruction from RAM using the program counter register to track the current position.

2

Decode

The instruction is decoded by the control unit to determine what operation needs to be performed and which operands to use.

3

Execute

The ALU (Arithmetic Logic Unit) performs the operation — addition, comparison, bitwise logic, memory read/write.

4

Write Back

The result is stored back to a register or RAM. The program counter advances and the cycle repeats.

5

Cache & Pipeline

Modern CPUs run multiple instructions simultaneously via pipelining and store frequently used data in L1/L2/L3 cache for nanosecond access.

Timeline

History of Computing

From room-sized machines to devices smaller than a fingernail — 80 years of relentless innovation.

1945
ENIAC — The First General-Purpose Computer
Weighed 30 tons, consumed 150 kW of power, and filled a 1,800 sq ft room. Contained 18,000 vacuum tubes and could perform 5,000 additions per second. Programmed by physically rewiring patch cables.
1958
The Integrated Circuit
Jack Kilby (Texas Instruments) and Robert Noyce (Fairchild) independently invented the IC, shrinking entire circuits onto a single chip. This invention triggered Moore's Law and the modern semiconductor industry.
1971
Intel 4004 — The First Microprocessor
A complete CPU on a single chip, running at 740 kHz with 2,300 transistors. Today's CPUs contain over 100 billion transistors — 43 million times more — and run 7,000× faster.
1984
Apple Macintosh — The GUI Revolution
The first mass-market computer with a graphical user interface and mouse. Made computing accessible to non-engineers and defined the personal computer paradigm for decades.
1991
Linux & the World Wide Web
Linus Torvalds released Linux 0.01. Tim Berners-Lee launched the World Wide Web. Two events that would reshape global computing infrastructure — Linux now runs 97% of the world's servers.
2007
iPhone — The Smartphone Era
Apple's iPhone combined a phone, internet browser, and touchscreen computer. Today over 6 billion people carry a computer more powerful than the machines that sent astronauts to the Moon.
2020
Apple M1 — ARM Conquers the Desktop
Apple's custom ARM chip shattered x86 performance-per-watt benchmarks, proving that RISC architecture could dominate desktop computing and pushing Intel and AMD to dramatically accelerate their roadmaps.
2022–Now
The AI Computing Revolution
Large language models and diffusion models drove an unprecedented demand for GPU compute. Nvidia's H100 became the most sought-after chip in history. AI accelerators are now standard in phones, laptops, and data centers.
What's Next

The Future of Computing

The next frontiers that will redefine what computers can do.

⚛️ Quantum Computing

Quantum computers use qubits that exist in superposition, enabling them to solve certain problems exponentially faster than classical computers. IBM, Google, and startups race toward fault-tolerant quantum advantage.

🧠 Neuromorphic Chips

Brain-inspired chips like Intel's Loihi process information using spiking neural networks — consuming 1000× less power than conventional AI accelerators for inference tasks.

📦 3D Chip Stacking

Instead of shrinking transistors further, manufacturers are stacking chips vertically. AMD's 3D V-Cache and HBM memory dramatically reduce latency by placing memory directly on top of the processor die.

💡 Photonic Computing

Using light instead of electricity to transmit and process data. Photonic chips could enable data center interconnects and AI inference at the speed of light with near-zero energy loss.

🌐 Edge & Ambient AI

AI inference moving to the device itself — phones, cameras, sensors. No cloud latency, full privacy. Every device becomes intelligent with dedicated NPUs processing data locally in real time.

🔗 Brain-Computer Interfaces

Companies like Neuralink are developing implantable chips that read neural signals directly from the brain, allowing humans to control computers with thought — and potentially restoring lost motor function.

Reference

Key Terms Glossary

Essential vocabulary for understanding computer hardware and software.

Transistor
A microscopic switch that forms the basic building block of all digital chips. Modern CPUs contain over 100 billion transistors.
Clock Speed (GHz)
The number of cycles a CPU executes per second. 5 GHz = 5 billion cycles per second. Higher frequency = more instructions per second.
TDP (Thermal Design Power)
The maximum heat a cooling solution must dissipate. A 125W TDP CPU requires a cooler rated for at least 125W.
Latency
The delay before data transfer begins. L1 cache: ~1ns. RAM: ~100ns. SSD: ~100μs. HDD: ~10ms. Network: ~1-100ms.
Throughput
The amount of data processed per unit of time. Measured in GB/s for storage, Gbps for networks, TFLOPS for GPUs.
BIOS / UEFI
Firmware stored on the motherboard that initializes hardware during boot and hands control to the operating system.
Virtualization
Running multiple operating systems simultaneously on one machine using a hypervisor (e.g. Proxmox, VMware, Hyper-V).
Process Node (nm)
The size of transistors on a chip. Smaller = more transistors, less power, better performance. Current leaders: 2–3nm.
PCIe
Peripheral Component Interconnect Express — high-speed bus connecting GPUs, NVMe drives, and NICs to the CPU. PCIe 5.0 = 32 GT/s per lane.
Software Foundation

How Operating Systems Work

The OS is the invisible layer that transforms bare hardware into a usable, multitasking machine.

1

Kernel

The core of the OS. Runs in privileged mode with direct hardware access. Manages CPU scheduling, memory, device drivers, and system calls. Linux, Windows NT, and XNU (macOS) are all monolithic or hybrid kernels.

2

Process & Thread Management

The OS scheduler decides which process runs on which CPU core at any moment. Context switching lets hundreds of programs appear to run simultaneously, switching between them thousands of times per second.

3

Virtual Memory

Every process believes it owns the entire address space. The OS maps virtual addresses to physical RAM pages using a page table. If RAM is full, pages are swapped to disk (swap/pagefile).

4

File System

Organizes data on storage into files and directories. Linux uses ext4, btrfs, or ZFS. Windows uses NTFS. macOS uses APFS. Each handles permissions, journaling, and snapshots differently.

5

System Calls

Programs communicate with the kernel via system calls — open(), read(), write(), fork(), exec(). These are the API between user-space apps and the hardware-facing kernel.

Fundamentals

Binary & How Computers Process Data

Every photo, video, message, and program is ultimately just ones and zeros — electrical signals that are either on or off.

B

Binary (Base-2)

Computers use binary because transistors have two states: ON (1) and OFF (0). The number 42 in binary is 00101010. Eight bits = 1 byte = 256 possible values.

H

Hexadecimal (Base-16)

Humans use hex as shorthand for binary. Each hex digit represents 4 bits. The color #3B82F6 encodes red=59, green=130, blue=246 — all as two hex digits each.

A

ASCII & Unicode

Text is stored as numbers. ASCII maps 128 characters to numbers 0–127. Unicode (UTF-8) extends this to over 1 million characters, supporting every language, emoji, and symbol on Earth.

L

Logic Gates

AND, OR, NOT, XOR, NAND — transistors wired together form logic gates. Millions of gates form adders, multiplexers, and registers. An ALU is built entirely from these basic gates doing binary arithmetic.

C

Compression & Encoding

Raw data is compressed to save space. JPEG reduces photos 10:1 using DCT math on pixel blocks. MP3 removes audio frequencies humans can't hear. H.265 encodes video with 50% less size than H.264.

Deep Dive

Networking: TCP/IP, DNS & HTTP

Every time you open a website, a precisely choreographed sequence of protocols fires off in milliseconds.

1

DNS — The Phone Book

You type google.com. Your OS asks a DNS resolver to translate that name to an IP address (e.g. 142.250.74.46). DNS is a globally distributed database with billions of records, answering ~1 trillion queries per day.

2

TCP — Reliable Delivery

A TCP connection is established via a 3-way handshake: SYN → SYN-ACK → ACK. TCP guarantees packets arrive in order and retransmits lost ones. Every byte of an HTTP response is acknowledged.

3

TLS — Encryption

Before HTTP data flows, TLS negotiates encryption keys using asymmetric cryptography. Your browser and the server agree on a shared secret so all traffic is encrypted — making HTTPS private and tamper-proof.

4

HTTP/3 — The Latest Protocol

HTTP/3 replaces TCP with QUIC (UDP-based), eliminating head-of-line blocking. A lost packet only delays its own stream, not all streams. Page loads are measurably faster on high-latency connections.

5

IP Routing

The Internet Protocol routes packets across networks. Each router examines the destination IP and forwards to the next hop. A packet from London to Tokyo may traverse 15+ routers in under 200ms.

Infrastructure

Data Centers & Cloud Computing

The internet runs on millions of servers packed into massive, energy-hungry facilities spread across the globe.

🏢

What is a Data Center?

A facility housing thousands of servers in racks, connected by high-speed fiber and powered by redundant UPS systems and diesel generators. The largest hyperscale data centers consume over 1 GW of power — more than a small city.

☁️

Cloud Providers

AWS, Azure, and Google Cloud rent compute, storage, and networking on-demand. You pay only for what you use. A startup can access the same infrastructure as a Fortune 500 company in seconds.

📦

Containers & Kubernetes

Docker packages apps with all their dependencies into portable containers. Kubernetes orchestrates thousands of containers across hundreds of servers, auto-scaling to demand and self-healing on failure.

⚡

CDN — Content Delivery Networks

Cloudflare, Akamai, and Fastly cache content on servers close to users. This page is delivered through Cloudflare's global network — your request never travels to the origin server, it hits an edge node near you.
