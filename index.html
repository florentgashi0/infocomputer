💻 Computers

A guide to how computers work, their history, and key components

🪨

What is a Computer?

A computer is an electronic device that processes data according to a set of instructions (programs). It can perform arithmetic, logic, input/output, and storage operations at extremely high speed.

⚙️

CPU — The Brain

The Central Processing Unit executes instructions. Modern CPUs have multiple cores allowing parallel processing.

  • Clock Speed GHz
  • Cores 2 – 192+
  • Cache L1 / L2 / L3
  • Top Makers Intel, AMD, Apple
💾

RAM — Memory

Random Access Memory is fast, temporary storage used while programs run. More RAM means more applications can run simultaneously without slowdowns.

  • Types DDR4, DDR5, LPDDR5
  • Typical Range 8 GB – 128 GB
  • Speed 3200 – 6400 MT/s
📡

Storage

Persistent storage keeps data when power is off. SSDs have largely replaced HDDs for speed, while HDDs remain for bulk storage.

  • HDD Magnetic, slow, cheap
  • SSD Flash, fast, common
  • NVMe PCIe, very fast
  • Interface SATA / PCIe / USB
🎮

GPU — Graphics

The Graphics Processing Unit renders images and video. Modern GPUs are also used for AI, machine learning, and scientific computing due to their massively parallel architecture.

  • Top Makers Nvidia, AMD, Intel
  • VRAM 4 GB – 80 GB
  • Uses Gaming, AI, 3D, Video
⚡

Motherboard & Power

The motherboard connects all components. The PSU (Power Supply Unit) converts AC mains power to regulated DC voltages required by computer components.

  • Form Factors ATX, mATX, ITX
  • Chipsets Z790, X670, B650
  • PSU Ratings 80+ Bronze – Titanium
🌐

Networking

Computers connect to networks via Ethernet (wired) or Wi-Fi (wireless). Networking allows resource sharing, communication, and internet access.

  • Ethernet 100 Mbps – 10 Gbps
  • Wi-Fi 6 Up to 9.6 Gbps
  • Protocols TCP/IP, DNS, HTTP
💻

Operating Systems

The OS manages hardware and provides services for software. It acts as an intermediary between applications and the physical hardware.

  • Windows Most popular desktop
  • Linux Servers & power users
  • macOS Apple hardware
  • Android / iOS Mobile

A Brief History of Computers

1945

ENIAC — first general-purpose electronic computer, weighed 30 tons and filled an entire room.

1958

Invention of the integrated circuit (microchip) by Jack Kilby and Robert Noyce.

1971

Intel releases the 4004 — the first commercially available microprocessor.

1981

IBM launches the IBM PC, establishing the x86 architecture that dominates to this day.

1991

Linux kernel created by Linus Torvalds. The World Wide Web goes public.

2007

The iPhone launches, beginning the smartphone era and putting powerful computers in everyone's pocket.

2020s

AI accelerators and neural processing units (NPUs) become standard; quantum computing advances rapidly.
